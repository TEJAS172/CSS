defining color value

colors in css must often specific in the following formates:

  1. S COLOR keyword - like - "red","green","blue","transparent",etc
  2. A HEX value - like"#000", "#00ff",etc
  3. An RGB value (red,green,blue)- like "rgb(255,0,0)"
